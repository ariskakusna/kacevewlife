What is balayage/color melt/ombré? All these terms refer to a style that involves coloring techniques that take the hair from darker to lighter as you move from root to tip. Because the essence of the look (from dark to light) is the same for all of these styles, it's easy to think they're the same, but they're not. What is a hair colour melt? What is color melting? Color melting is a technique that is used to blend highlights or balayage into a client's root color in order to create a more gradual transition and subtler lines of demarcation.
